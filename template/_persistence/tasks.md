## Convención de ID

- Cada tarea tiene un ID estable: `T-001`, `T-002`, ...
- No reutilizar IDs de tareas eliminadas.
- Las subtareas se anidan bajo su tarea padre.

      Prioridad: alta | media | baja · Responsable: <agente/persona> · Ref: [[progress]]
